# Documentation of test.py

The code contains different functions for the different parts.

- The Parse function helps to open the given 'txt' document and fetch the relevant details from it.
- The V1 function moves the boxes one-by-one and contains a test to check every movement
- The V2 function moves multiple boxes together
- The Print_state function tries to show to output the same way as the input
- The Validate_moves function checks the syntaxes of the move commands
- Then in the "main", the movements starts

#### Start the code with "python test.py"
